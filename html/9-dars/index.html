Juft tag => h1, p, span, b, i ,u ...

span tegi => inline element, yonma-yon yozadi

b => tegi (bold) => qalin shriftda yozadi, inline element

u tegi => ostki chiziq bilan yozadi

i tegi => yotiq shriftda yozadi

button => tugma

Ul turlari => ( square, circle )
